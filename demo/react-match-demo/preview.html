API Integration

Endpoint: POST http://recon.artsdata.ca/match

Request Format:

{
  "queries": [{
    "type": "schema:Place",
    "limit": 10,
    "conditions": [{
      "matchType": "name",
      "propertyValue": "search term"
    }]
  }]
}

Response: Array of candidates with name, ID, description, score, and type information
